# Support Ticket System

Sistema web de soporte interno desarrollado con Laravel para la gestión de incidencias, seguimiento de tickets y control operativo por estados, técnicos y áreas.

## Características

- Gestión de tickets
- Panel administrativo
- Roles y permisos
- Dashboard operativo
- Control de estados
- Historial y seguimiento
- Adjuntos y comentarios
- Sistema de autenticación
- Interfaz responsive
- Flujo de atención técnica

## Stack

- Laravel 12
- PHP
- SQL Server
- Blade
- Tailwind CSS
- Vite
